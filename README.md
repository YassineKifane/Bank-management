# Bank Management

## Description

C ++ application allowing the management of a bank:

- Customer management.

  - Add new customer.
  
  - Modify customer informations.
  
  - Search customer informations.
  
  - Delete Customer.
  
- Account management.

  - Add New Account to a Customer.
  
  - Show Customer Balance.
  
  - Delete Customer account
  
- Operations management
  
  - Withdraw an amount
  
  - Deposit an amount
  
## Requirement

- GCC compilator
- C/C++ IDE (CodeBlock)
